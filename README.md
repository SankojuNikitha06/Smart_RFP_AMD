# Smart RFP – Agentic AI for Autonomous Bidding
- Overview : Smart RFP is a multi agent AI system designed to automate the end to end bidding workflow. It intelligently processes RFP documents, generates optimized bids, evaluates risks, and ensures accuracy through verification and self correction.

 # Key Features : 
🤖 Multi Agent Architecture for modular intelligence
⚡ Autonomous Workflow Execution
🔍 Error Detection & Self-Correction
📜 Comprehensive Audit Logging
📈 Scalable for Enterprise Use Cases
🏗️ System Architecture

# Agents Involved : 
- Orchestrator Agent – Manages workflow execution
- Retrieval Agent – Extracts and processes RFP data
- Pricing Agent – Generates optimal pricing strategies
- Risk Evaluation Agent – Assesses potential risks
- Verification Agent – Validates outputs and detects errors
- Submission Agent – Produces final bid output

# Workflow Pipeline - 
- Input → Retrieval → Pricing → Risk Evaluation → Verification → Output

# Setup Instructions

- git clone https://github.com/SankojuNikitha06/Smart_RFP_AMD

# Example Workflow
- Input RFP data

- Outputs the final verified result
